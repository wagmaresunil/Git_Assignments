to push the files from local to remote
we have 4 stages

working directory area ->local
adding/staging area->unstaged to staged
committing area->history
remote directory  ->GitHub repo

git init
git add .
git status
git commit -m "msg"
git config --global user.name "username"
git config --global user.email "useremail"
git commit -m "msg"
git remote add origin httpurl
git branch -M main
git push origin main
